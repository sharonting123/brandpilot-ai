# Changelog

本仓库采用 [语义化版本](https://semver.org/lang/zh-CN/)（`主版本.次版本.修订号`），并通过 **Git Tag** 标记每个可回滚版本。

| 版本 | Tag | 说明 |
|------|-----|------|
| 当前 | 见 `VERSION` 文件 | 线上/本地当前版本号 |

## 如何回滚

```bash
# 查看所有版本
git tag -l 'v*' --sort=-v:refname

# 回滚到某一版（只读查看）
git checkout v1.0.0

# 生产回滚：基于旧 tag 新建修复分支
git checkout -b hotfix/from-v1.0.0 v1.0.0
```

GitHub → **Releases** 页面可查看每个版本的变更摘要。
